# Huy Automation AI Website V20

V20 là bản nâng cấp từ V19 theo hướng bán hàng cho khách hàng doanh nghiệp:

- Không bán phần mềm rời rạc.
- Không đưa ngôn ngữ kỹ thuật ra tầng bán hàng.
- AI Employee là sản phẩm khách hàng nhìn thấy.
- Automation phía sau là bằng chứng hệ thống hoạt động.
- Build in Public là bằng chứng kiểm chứng nội bộ trước khi triển khai cho khách hàng.
- Gói AI Sales BĐS là flagship demo, không phải website bán bất động sản.

## Cách dùng

Giải nén, copy toàn bộ bên trong và paste đè vào thư mục project hiện tại.

```powershell
npm install
npm run dev
```

Sau khi kiểm tra ổn:

```powershell
git add .
git commit -m "Website V20 - AI Employee Sales Hub"
git push origin main
```
