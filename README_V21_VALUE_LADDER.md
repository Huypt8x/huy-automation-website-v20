# Huy Automation AI Website V21 - Value Ladder

Bản này giữ nguyên giao diện V21 hiện đại, nhưng đổi chiến lược bán hàng:

- Không bán 3 gói lớn ngay từ đầu.
- Bắt đầu bằng 1 AI Employee nhỏ từ 5 triệu.
- Khách bắt đầu với một AI Employee, sau đó mở rộng thêm Quế Anh, Hà Anh, Vân Anh, Hải Anh... khi doanh nghiệp sẵn sàng
- AI Business OS vẫn giữ nhưng là lộ trình mở rộng, không phải sản phẩm mở đầu.

Trang giá đã đổi sang:

1. AI Employee Mini - Từ 5 triệu
2. Quế Anh / AI Sales BĐS - Từ 9 triệu
3. AI Business OS - Liên hệ

Quy trình kiểm tra:

```powershell
npm run dev
npm run build
git add .
git commit -m "Website V21 value ladder"
git push
```
