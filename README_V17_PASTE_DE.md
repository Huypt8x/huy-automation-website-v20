# Huy Automation Website V17 - Paste đè

Bản V17 sửa theo hướng khách hàng dễ ra quyết định mua hơn.

## Đã thêm / sửa

- Active menu trên Header:
  - /automation/* và /employees/* sẽ active AI Employees.
  - /systems/* sẽ active AI Company OS.
  - /case-studies/* sẽ active Case Studies.
  - /consultation hoặc #pricing sẽ active Pricing.
- Workflow Detail Page theo format bán hàng:
  1. Hero + CTA xem video demo
  2. Before / After
  3. Workflow hoạt động thế nào
  4. Video demo popup
  5. ROI cho chủ doanh nghiệp
  6. Gói triển khai + giá tham khảo
  7. CTA Audit AI miễn phí / Đặt lịch Demo 15 phút
- Thêm component video popup:
  - src/components/VideoDemoButton.tsx

## Cách dùng

Giải nén ZIP, copy toàn bộ file/thư mục bên trong và paste đè vào:

J:\AI Automation\Website\huy-automation-website-v1

Sau đó chạy:

npm run dev
