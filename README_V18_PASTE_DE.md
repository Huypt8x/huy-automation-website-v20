# Huy Automation Website V18 - Paste đè

## Đã thêm

- Video: `public/media/videos/que-anh-g1-content-facebook.mp4`
- Trang Quế Anh: `/employees/que-anh#video`

  - Hiển thị video demo Quế Anh trong phần Demo.
- Cập nhật Quế Anh:
  - Vai trò: `G1 Content Facebook`
  - Phòng ban: `Marketing`
  - Thuộc hệ thống: `G1 Content OS`

## Cách dùng

Giải nén ZIP, copy toàn bộ file/thư mục bên trong và paste đè vào:

`J:\AI Automation\Website\huy-automation-website-v1`

Sau đó chạy:

```powershell
cd "J:\AI Automation\Website\huy-automation-website-v1"
npm run dev
```
